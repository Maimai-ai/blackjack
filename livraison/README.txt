Afin d'éxécuter notre Blackjack : 
	- Ouvrir un terminal(invite de commande) dans le repertoire "livraison", là où se situe le fichier README
	- Puis dans ce terminal ecrivez la commande suivante : "cd cartes"
	
	- 2 utilisations possibles pour lancer le blackjack :
		- Puis lancer la commande "ant run"
		- En lançant cette commande, elle génèrera une javadoc, un jar et compilera tous nos fichiers situer dans le repertoire "src" .java et affichera la fenetre contenant une vue d'une pioche et des cartes en votre possession
		- Cet commande crée une librairie dont depend notre blackjack
		- Ensuite fermé cette fenêtre

		- Si nous ne voulons pas ouvrir cette fenetre au lieu de lancer la commande "ant run", lancer la commande "ant dist"
	
	- Puis losqu'une des deux possibilitées a été exécuté, ecrivez toujours dans cet inviter de commande "cd .."
	- Puis lancé la, et ecrivez "cd blackjack"
	- et pour finir executez la commande "ant run" qui executera notre Blackjack
